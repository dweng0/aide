# #19 — Remove goals-archive.md

**Labels:** enhancement · done

## Problem

`note_organizer.sh` archived the previous day's `daily-goals.md` into a write-only `goals-archive.md` that grew unbounded and was never read back.

## Resolution

Instead of discarding or archiving to a separate file, goals are now appended to yesterday's daily note (`~/notes/YESTERDAY.md`) under a `## Goals` section each morning, then `daily-goals.md` is cleared. Historical goals are preserved inline alongside the journal entry for that day, enabling future metrics without a separate growing file.
